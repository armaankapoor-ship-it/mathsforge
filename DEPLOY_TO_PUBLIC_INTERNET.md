# Deploy VectorForge to the Public Internet for Free

Use GitHub Pages. It is free and works with this static frontend project.

Quick path:

1. Create a free GitHub account.
2. Create a new public repository.
3. Upload this full project folder.
4. Go to `Settings -> Pages`.
5. Select `GitHub Actions`.
6. Run the included deployment workflow.
7. Share the final link:

```text
https://your-github-username.github.io/your-repository-name/
```

No paid hosting, paid domain, backend, database, or API key is needed.
