# MathsForge

Main launcher for the separate Class 12 MathsForge chapter websites.

Netlify website name:

```text
mathsforge
```

Netlify public website:

```text
https://mathsforge.netlify.app/
```

## Chapter Websites

- [Relations and Functions Forge](https://relations-functions-forge.netlify.app/)
- [Inverse Trigonometric Functions Forge](https://inverse-trigonometric-functions-forge.netlify.app/)
- [Matrices Forge](https://matrices-forge.netlify.app/)
- [Determinants Forge](https://determinants-forge.netlify.app/)
- [Continuity and Differentiability Forge](https://continuity-differentiability-forge.netlify.app/)
- [Application of Derivatives Forge](https://application-of-derivatives-forge.netlify.app/)

- [Linear Programming Forge](https://linear-programming-forge.netlify.app/)
- [Probability Forge](https://probability-forge.netlify.app/)

Each chapter is a standalone static website with MathsForge-style design, GitHub Pages compatibility, Netlify config, and 20 chapter-specific 3D models.

## How to Publish This Launcher for Free Using Netlify

1. Create a free Netlify account at `https://www.netlify.com`.
2. Click **Add new site**.
3. Choose **Deploy manually**.
4. Drag the `chapter-websites` folder into Netlify.
5. Set the site name to `mathsforge`.
6. Copy the public link:

```text
https://mathsforge.netlify.app/
```
